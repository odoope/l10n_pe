REQUERIMIENTOS MÍNIMOS

El módulo incluye las tablas de SUNAT indispensables para la integración con la facturación electrónica en Perú. Añade nuevos campos en la factura para efectos de cáculos de impuesto y descuentos. Además se ha creado un apartado para gestionar "Tiendas", con esto podremos gestionar las series de todos nuestros documentos de pago.

Nuevos campos

Ahora podrás obtener datos del IGV, venta gratuita y descuentos en la factura.

Tiendas / Locales

Gestione sus comprobantes, series y usuarios por cada tienda o local.

Tablas SUNAT

Incluye las tablas SUNAT necesarios para la gestión de la factura electrónica.

Nuevas características!

  • Tipos de comprobantes electrónicos: Boleta de venta, Factura de venta y notas de crédito electrónica.
  • Nuevos campos en la factura para el cálculo de: IGV, venta exonerada, venta inafecta, venta gratuita, descuentos por item y descuento global.
  • Gestión de tiendas y/o locales de venta para el seguimiento y control de las serias de cada comprobante electrónico.
  • Tipos de impuesto para diferenciar IGV, ISC, exonerado o infacto.
  • Nuevas tablas para la gestión de los catálogos SUNAT.
